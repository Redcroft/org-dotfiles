** MBSync
*** Redcroft
#+begin_src conf :tangle "~/.mbsyncrc"
  IMAPAccount redcroft
  Host imap.hostinger.com
  Port 993
  User richard@redcroft.tech
  PassCmd "pass hostinger.com/richard@redcroft.tech"
  SSLType IMAPS

  IMAPStore redcroft-remote
  Account redcroft

  MaildirStore redcroft-local
  SubFolders Verbatim
  Path ~/.mail/redcroft/
  Inbox ~/.mail/redcroft/Inbox
  Trash ~/.mail/redcroft/Trash

  Channel redcroft
  Far :redcroft-remote:
  Near :redcroft-local:
  Patterns *
  SyncState *
  Create Both
  Expunge Both
  CopyArrivalDate yes
  Sync All
#+end_src
